## AI Prompt for Production Implementation

```
You are building a receipt OCR extraction system. Given a receipt image, extract structured data with the following requirements:

INPUT: Receipt image (JPG/PNG)

OUTPUT FORMAT:
{
  "confidence": float (0-1),
  "date": "MM/DD/YY",
  "time": "HH:MM:SS",
  "item_count": integer,
  "store": {
    "name": string,
    "address": string,
    "phone": string,
    "store_number": string
  },
  "items": [
    {
      "name": string,
      "sku": string,
      "unit_price": string,
      "quantity": integer,
      "total_price": string,
      "tax_code": string,
      "category": string (optional)
    }
  ],
  "totals": {
    "subtotal": string,
    "tax": string,
    "total": string,
    "cash": string (optional),
    "change": string (optional)
  },
  "payment_details": {
    "payment_type": string,
    "reference_number": string (optional)
  },
  "receipt_number": string
}

PROCESSING STEPS:
1. Image Preprocessing:
   - Convert to grayscale
   - Deskew/rotate if needed
   - Denoise using fastNlMeansDenoising
   - Increase contrast with CLAHE
   - Apply Otsu binarization
   - Upscale 2x if low resolution

2. OCR Extraction:
   - Primary: PaddleOCR or Google Cloud Vision API
   - Fallback: EasyOCR with optimized parameters
   - Extract text with bounding boxes and confidence scores

3. Layout Analysis:
   - Sort text by Y-coordinate (top to bottom)
   - Identify regions: header (store info), body (items), footer (totals)
   - Group text by horizontal alignment

4. Structured Parsing:
   - Store info: First 3-5 lines, extract name/address/phone
   - Items: Lines matching pattern [NAME] [SKU] [PRICE]
   - Totals: Last 5-10 lines, match keywords (SUBTOTAL, TAX, TOTAL)
   - Use regex patterns:
     * Date: \d{2}/\d{2}/\d{2,4}
     * Time: \d{1,2}:\d{2}:\d{2}
     * Price: \d+\.\d{2}
     * SKU: \d{12,14}

5. Validation:
   - Sum item prices should equal subtotal
   - Subtotal + tax should equal total
   - Item count should match actual items parsed
   - All critical fields present (store name, date, total)
   - Flag for manual review if validation fails

6. Error Correction:
   - Common OCR errors: O→0, l→1, S→5 in numeric contexts
   - Fix impossible dates (e.g., month > 12)
   - Correct malformed prices (e.g., missing decimal)

QUALITY THRESHOLDS:
- Confidence < 0.7: Flag for manual review
- Missing totals: FAIL
- Math validation error > 0.05: Flag for review
- Item count = 0: FAIL

EDGE CASES:
- Faded thermal receipts: Apply aggressive contrast enhancement
- Rotated images: Use angle detection and rotation
- Multiple receipts: Process each separately
- Non-English text: Detect language first
```

## Recommended Implementation Architecture

### Option 1: Cost-Effective (Open Source)

```python
import cv2
import numpy as np
from paddleocr import PaddleOCR
import re
import json

class ReceiptOCR:
    def __init__(self):
        self.ocr = PaddleOCR(
            use_angle_cls=True,
            lang='en',
            use_gpu=False,
            show_log=False
        )
    
    def preprocess(self, image_path):
        """Enhanced preprocessing for receipt images"""
        img = cv2.imread(image_path)
        
        # Upscale if too small
        h, w = img.shape[:2]
        if w < 1000:
            scale = 1000 / w
            img = cv2.resize(img, None, fx=scale, fy=scale, 
                           interpolation=cv2.INTER_CUBIC)
        
        # Convert to grayscale
        gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
        
        # Denoise
        denoised = cv2.fastNlMeansDenoising(gray, h=10)
        
        # CLAHE for contrast
        clahe = cv2.createCLAHE(clipLimit=3.0, tileGridSize=(8,8))
        enhanced = clahe.apply(denoised)
        
        # Sharpen
        kernel = np.array([[-1,-1,-1], [-1,9,-1], [-1,-1,-1]])
        sharpened = cv2.filter2D(enhanced, -1, kernel)
        
        # Threshold
        _, binary = cv2.threshold(sharpened, 0, 255, 
                                 cv2.THRESH_BINARY + cv2.THRESH_OTSU)
        
        return binary
    
    def extract(self, image_path):
        """Extract structured data from receipt"""
        # Preprocess
        preprocessed = self.preprocess(image_path)
        
        # OCR
        results = self.ocr.ocr(preprocessed, cls=True)
        
        # Parse results
        parsed = self.parse_results(results)
        
        # Validate
        validated = self.validate(parsed)
        
        return validated
    
    def parse_results(self, ocr_results):
        """Convert OCR output to structured format"""
        lines = []
        for line in ocr_results[0]:
            bbox, (text, confidence) = line
            y_coord = bbox[0][1]
            lines.append({
                'text': text,
                'confidence': confidence,
                'y': y_coord
            })
        
        # Sort by Y coordinate
        lines.sort(key=lambda x: x['y'])
        
        # Extract fields
        data = {
            'store': self.extract_store_info(lines[:10]),
            'items': self.extract_items(lines),
            'totals': self.extract_totals(lines[-15:]),
            'date': self.extract_date(lines),
            'time': self.extract_time(lines)
        }
        
        return data
    
    def extract_store_info(self, header_lines):
        """Extract store information from header"""
        store = {}
        for line in header_lines:
            text = line['text'].lower()
            if 'walmart' in text:
                store['name'] = 'Walmart'
            elif re.search(r'\d{3}[-.]?\d{3}[-.]?\d{4}', line['text']):
                store['phone'] = line['text']
            elif re.search(r'\d+\s+\w+.*\d{5}', line['text']):
                store['address'] = line['text']
        return store
    
    def extract_items(self, lines):
        """Extract line items"""
        items = []
        for line in lines:
            # Check if line has price pattern
            price_match = re.findall(r'\b\d+\.\d{2}\b', line['text'])
            # Check if line has SKU pattern
            sku_match = re.findall(r'\b\d{12,14}\b', line['text'])
            
            if price_match and sku_match:
                # This is likely an item line
                item = {
                    'name': self.extract_item_name(line['text']),
                    'sku': sku_match[0],
                    'total_price': price_match[-1],
                    'quantity': 1
                }
                items.append(item)
        
        return items
    
    def extract_totals(self, footer_lines):
        """Extract totals from footer"""
        totals = {}
        for line in footer_lines:
            text = line['text'].lower()
            price = re.findall(r'\d+\.\d{2}', line['text'])
            
            if price:
                if 'subtotal' in text:
                    totals['subtotal'] = price[-1]
                elif 'tax' in text:
                    totals['tax'] = price[-1]
                elif 'total' in text and 'subtotal' not in text:
                    totals['total'] = price[-1]
                elif 'cash' in text or 'tend' in text:
                    totals['cash'] = price[-1]
                elif 'change' in text:
                    totals['change'] = price[-1]
        
        return totals
    
    def validate(self, data):
        """Validate extracted data"""
        confidence = 1.0
        notes = []
        
        # Check if subtotal + tax = total
        if all(k in data['totals'] for k in ['subtotal', 'tax', 'total']):
            subtotal = float(data['totals']['subtotal'])
            tax = float(data['totals']['tax'])
            total = float(data['totals']['total'])
            
            if abs((subtotal + tax) - total) > 0.05:
                confidence *= 0.7
                notes.append("Math validation failed")
        
        # Check for required fields
        if not data['store'].get('name'):
            confidence *= 0.5
            notes.append("Store name missing")
        
        if not data['totals'].get('total'):
            confidence *= 0.3
            notes.append("Total missing - CRITICAL")
        
        data['confidence'] = confidence
        data['notes'] = notes
        data['item_count'] = len(data['items'])
        
        return data

# Usage
ocr = ReceiptOCR()
result = ocr.extract('receipt.jpg')
print(json.dumps(result, indent=2))
```

### Option 2: Hybrid (Best Accuracy)

```python
class HybridReceiptOCR:
    def __init__(self, use_cloud=True):
        self.paddle = PaddleOCR(use_angle_cls=True, lang='en')
        self.use_cloud = use_cloud
        
        if use_cloud:
            from google.cloud import vision
            self.cloud_client = vision.ImageAnnotatorClient()
    
    def extract(self, image_path):
        # Try PaddleOCR first
        paddle_result = self.extract_paddle(image_path)
        
        # Validate
        if self.is_high_confidence(paddle_result):
            return paddle_result
        
        # Fallback to cloud API for low confidence
        if self.use_cloud:
            cloud_result = self.extract_cloud(image_path)
            return self.merge_results(paddle_result, cloud_result)
        
        return paddle_result
    
    def is_high_confidence(self, result):
        return (
            result['confidence'] > 0.8 and
            result.get('totals', {}).get('total') and
            len(result.get('items', [])) > 0
        )
    
    def extract_cloud(self, image_path):
        """Use Google Cloud Vision for difficult receipts"""
        with open(image_path, 'rb') as f:
            content = f.read()
        
        image = vision.Image(content=content)
        response = self.cloud_client.text_detection(image=image)
        
        # Parse Google's response
        return self.parse_cloud_response(response)
```

### Option 3: Claude API (Highest Accuracy, Easiest)

```python
import anthropic
import base64
import json

class ClaudeReceiptOCR:
    def __init__(self, api_key):
        self.client = anthropic.Anthropic(api_key=api_key)
    
    def extract(self, image_path):
        with open(image_path, 'rb') as f:
            image_data = base64.b64encode(f.read()).decode()
        
        prompt = """Extract all information from this receipt in JSON format.

Required fields:
- store: {name, address, phone, store_number}
- date: MM/DD/YY format
- time: HH:MM:SS format
- items: [{name, sku, unit_price, quantity, total_price, tax_code}]
- totals: {subtotal, tax, total, cash, change}
- receipt_number: transaction/receipt ID

Ensure all prices are strings with 2 decimal places.
Validate that subtotal + tax = total."""
        
        message = self.client.messages.create(
            model="claude-sonnet-4-5-20250929",
            max_tokens=4096,
            messages=[{
                "role": "user",
                "content": [
                    {
                        "type": "image",
                        "source": {
                            "type": "base64",
                            "media_type": "image/jpeg",
                            "data": image_data
                        }
                    },
                    {"type": "text", "text": prompt}
                ]
            }]
        )
        
        # Parse Claude's response
        response_text = message.content[0].text
        
        # Extract JSON from response
        json_match = re.search(r'\{.*\}', response_text, re.DOTALL)
        if json_match:
            return json.loads(json_match.group())
        
        return {"error": "Failed to parse JSON response"}

# Usage
ocr = ClaudeReceiptOCR(api_key="your-key")
result = ocr.extract('receipt.jpg')
```

## Final Recommendations

### For Production Use:

1. **Best Overall:** **Hybrid Approach**
   - Use PaddleOCR for 80% of receipts (good quality)
   - Fallback to Claude API for challenging cases (faded, skewed, damaged)
   - Cost: ~$0.01-0.03 per receipt (mostly PaddleOCR = free)
   - Accuracy: 93-97%

2. **Budget Option:** **PaddleOCR Only**
   - Cost: Free (open source)
   - Accuracy: 85-92% with good preprocessing
   - Speed: Fast (local processing)

3. **Premium Option:** **Claude API Primary**
   - Cost: ~$0.03-0.05 per receipt
   - Accuracy: 95-98%
   - Development time: Minimal
   - Best for MVP or low volume
