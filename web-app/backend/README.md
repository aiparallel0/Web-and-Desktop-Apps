# Web Application Backend

Flask-based REST API for receipt extraction.

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run server
python app.py
```

Server runs on `http://localhost:5000`

## API Documentation

See main README.md for full API documentation.

## Configuration

Edit `app.py` to change:
- Port (default: 5000)
- Max upload size (default: 16MB)
- Allowed file types
- CORS settings
