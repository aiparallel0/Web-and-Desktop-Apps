Component Features

  • ✅ Shows "✓ Selected" badge on selected available model
  • ✅ Shows "⚠ Dependencies Missing" badge on unavailable models
  • ✅ Grays out unavailable models with reduced opacity
  • ✅ Displays missing dependency list in red box
  • ✅ Prevents clicking on unavailable models
  • ✅ Shows tooltip with missing deps on hover
  • ✅ Prevents extraction if selected model becomes unavailable
