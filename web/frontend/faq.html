Getting Started

Getting started is easy:

  1. Visit our homepage and upload your first receipt - no signup required!
  2. For more features, create a free account by clicking "Sign In" and then "Register"
  3. Choose a subscription plan from our pricing page if you need more extractions
  4. Start uploading receipts and let our AI do the work

The free tier gives you 10 receipts per month with basic OCR models.

We offer 7 different OCR models. Here's when to use each:

  • Tesseract: Fast and reliable for clear, high-quality receipts
  • EasyOCR: Great for multilingual receipts and diverse fonts
  • PaddleOCR: Excellent for production use, especially Asian languages (Pro+ plans)
  • Florence-2: Advanced AI model for challenging or damaged receipts (Pro+ plans)
  • Spatial Multi-Method: Combines multiple models for maximum accuracy

For best results, try the Spatial Multi-Method model which uses ensemble techniques.

We support all common image formats:

  • JPEG/JPG
  • PNG
  • BMP
  • TIFF/TIF
  • PDF (single page receipts)

Maximum file size: 10MB per receipt. For batch processing, you can upload up to 100 receipts at once (Pro+ plans).

Troubleshooting

If processing fails, try these steps:

  1. Check image quality: Ensure the receipt is clear and well-lit
  2. Try a different model: Some models work better for certain receipt types
  3. Reduce file size: Large files (>10MB) may timeout
  4. Check format: Verify your file is in a supported format
  5. Use preprocessing: Enable image enhancement in advanced settings

Still having issues? Contact support at support@receiptextractor.com with your receipt ID.

Processing time depends on several factors:

  • Model choice: Tesseract is fastest (~2s), Florence-2 can take ~30s
  • Image size: Larger images take longer to process
  • Server load: Peak times may have slight delays
  • Preprocessing: Image enhancement adds processing time

Average processing times:

  • Tesseract: 2-5 seconds
  • EasyOCR: 5-10 seconds
  • Florence-2: 15-30 seconds
  • Spatial Multi-Method: 20-40 seconds
